# Changelog

## [1.0.0] - 2026-03-07

### Added
- SSH terminal with xterm.js integration
- SFTP sidebar with file browser, drag & drop, chmod, rename, copy path, new file
- Remote file editor (download, edit, re-upload)
- Local terminal support (PowerShell, Bash, CMD)
- WSL and Docker container attach
- Telnet, Serial/COM port, FTP sessions
- RDP and VNC launch support
- S3 browser (bucket/object browse, upload/download)
- SSH key generator and tunnel manager
- SSH agent forwarding and proxy jump
- SSH config auto-import (~/.ssh/config)
- Session save/load/delete with groups and tags
- Connection templates and environment variable presets
- Session import/export
- Auto-login with credential lookup
- Auto-reconnect on SSH disconnect
- Quick Connect bar (Ctrl+K)
- Command Palette (Ctrl+P)
- Terminal split view (horizontal/vertical)
- Terminal broadcast mode (input to multiple tabs)
- Clipboard history (Ctrl+Shift+V)
- Port scanner
- Password manager with AES-256-GCM encryption
- Remote system monitoring (CPU, memory, load sparklines in status bar)
- 15+ themes (Catppuccin, Dracula, Nord, Tokyo Night, GitHub, Solarized, Gruvbox, etc.)
- Live terminal theme switching
- Settings persistence (localStorage)
- Pinned Home tab with connection hub
- Tab drag & drop reorder
- Tab color customization
- OS brand icons for detected remote systems
- Connection health check
- Session logging (start/stop per session)
- Content Security Policy (CSP)
- Custom frameless window with title bar controls
